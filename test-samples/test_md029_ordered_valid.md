# MD029 Valid Cases for "ordered" style

All ordered lists should use incrementing numbers.

## Valid list starting with 1

1. First item
2. Second item  
3. Third item

## Valid list starting with 0

0. First item
1. Second item
2. Third item

## Large numbers

100. Item hundred
101. Item hundred-one
102. Item hundred-two

## Zero-padded numbers

08. Item eight
09. Item nine
10. Item ten

## Single item lists (always valid)

1. Single item

## Another section

42. Another single item

## Nested lists

1. Outer item
   1. Inner item
   2. Inner item
2. Outer item
   0. Different inner style
   1. Continues
3. Outer continues
