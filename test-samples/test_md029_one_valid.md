# MD029 Valid Cases for "one" style

All ordered lists should use "1." for every item.

## Valid list

1. First item
1. Second item
1. Third item

## Another valid list

1. Different list
1. Can still use all 1s
1. Even in separate sections

## Single item list

1. Single item

## Nested lists

1. Outer item
   1. Inner item
   1. Inner item
1. Outer item
